UsbTerminal

Welcome to v2.0

UsbTerminal was updated to version 2.0. The last update of UsbTerminal v1.x was way back in 2016. Version 2.0 is a complete re-write of the app. It has a new look, a new (flat) app icon, and many new features.
Some of these new features are:
  • Support for some VT100 escape-codes
  • Display data as text or as hex-dump
  • Support of additional device types
  • Support of text coloring
  • Support of communication in the background even when using other apps
UsbTerminal is now open-source. Its source code can be found in https://github.com/liorhass/UsbTerminal.

For help on how to use this new version of the app, please go to the "Help" page (accessible from the navigation-drawer)

NOTE: One issue with this update is that the directory where the app stores log-files has changed. Unfortunately due to security and privacy enhancements in Android, the app can no longer access old log-files that were generated with older versions.
You should be able to access these files with a file manager app such as "Files by Google". Look for a directory named "usbterminal" under "Internal storage" or an SD card if you have any, or under "/storage/emulated/0/usbterminal".
I'm sorry for this inconvenience.
